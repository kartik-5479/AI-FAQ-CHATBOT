## 💡 How It Works

### Data Flow
```
User Question
    ↓
[NLP Preprocessing]
    ↓
[TF-IDF Vectorization]
    ↓
[Cosine Similarity Matching] (searches 165 FAQs)
    ↓
[Best Match Selection]
    ↓
[Confidence Scoring]
    ↓
Bot Response + Confidence %
```

### Database Schema
```sql
faqs (165 records, 22 categories)
├── question (unique)
├── answer (detailed)
├── category (new categories included)
├── keywords (optimized for each topic)
├── created_at
└── updated_at

chat_history (tracks all conversations)
├── user_question
├── bot_answer
├── confidence
├── intent
├── faq_id (foreign key to faqs)
└── timestamp

favorites (bookmarked Q&As)
├── chat_id
├── user_question
├── bot_answer
└── date_saved

statistics (automatically updated)
├── total_questions
├── total_conversations
├── avg_confidence
├── most_asked_category
└── last_updated
```
