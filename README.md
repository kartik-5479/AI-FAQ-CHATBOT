## 📖 About The Project

JARVIS (Just A Rather Very Intelligent System) is a modern AI-powered FAQ chatbot developed using Python, Flask, and Natural Language Processing techniques.

The chatbot analyzes user questions, processes text using NLP, and retrieves the most relevant answer from a large FAQ knowledge base. It utilizes TF-IDF Vectorization and Cosine Similarity algorithms to identify the best matching response and provides confidence scores for every answer.

The project also includes voice interaction, chat history management, favorites, analytics dashboard, dark mode, and data export functionality.
